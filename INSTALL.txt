Σε αυτή την ενότητα θα αναλύσουμε πληροφοριές σχετικά με την εγκατάσταση ου εικονικού εργαστηρίου που έχουμε φτιάξει:

Αρχικά κάνουμε git clone το https://github.com/andreasdimizas/cloud_project.git για την κλωνοποίηση του αποθετηρίου Git σε έναν τοπικό φάκελο στον υπολογιστή μας.Αυτή η εντολή αντιγράφει όλο το ιστορικό του κώδικα στον υπολογιστή μας,και μας δημιουργεί έναν τοπικό κλώνο που μπορούμε να εργαστούμε πάνω σε αυτό.

1.cd /cloud_project: Αλλάζει τον τρέχοντα κατάλογο στο /cloud_project.

2.Κάνουμε edit το αρχείο cloud_project/myblog/config.toml και τοποθετούμε στο base_url το domain name μας.

3.Κάνουμε edit το αρχείο cloud_project/site.conf/site1.conf και βάζουμε τον nginx το δικό μας domain

4.Μεταβαίνουμε στον κατάλογο cloud_project και εκτελούμε το αρχείο cloud_project/run.sh
