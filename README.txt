Ένα εργαλείο που έχουμε χρηστιμοποιήσει για την εργασία ειναι το Zola το οποίο είναι ένα εργαλείο στατικής δημιουργίας ιστοσελίδων που μας επιτρέπει να δημιουργήσουμε γρήγορα και εύκολα στατικές ιστοσελίδες.

Επίσης μια άλλη υπηρεσία που έχει χρησιμοποιηθεί είναι το YouTube Local, που είναι ένα εργαλείο που μας παρέχει τη δυνατότητα να χρησιμοποιήσουμε ανώνυμα την υπηρεσία YouTube

Επιπρόσθετα ακόμη μια υπηρεσία ειναι το SearXNG το οποίο συγκεντρώνει αποτελέσματα αναζήτησης από πολλές δημοφιλείς μηχανές αναζήτησης, επιτρέποντάς μας να προβάλλουμε αποτελέσματα από διάφορες πηγές σε ένα μέρος.

Επίσης έχουμε χρησιμοποήσει και ngnix όπου στην δική μας εργασία ο nginx χρησιμοποιείται σάν server της δικής μας σελίδα,και λειτουργεί σαν reverse proxy για τις άλλες υπηρεσίες.
